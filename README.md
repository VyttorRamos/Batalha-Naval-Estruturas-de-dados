# 📚 Estruturas de Dados - Trabalho Prático

Este projeto consiste na implementação de diversas **Estruturas de Dados** e **Jogos de Lógica em C++**, desenvolvidos como parte da disciplina de Estrutura de Dados.

---

## 🚀 Funcionalidades

O programa possui um **menu interativo**, permitindo explorar diferentes implementações:

### 🎮 Jogos

**🟢 Jogo da Velha**  
Clássico jogo para dois jogadores (X e O), em um tabuleiro 3x3, com:
- Verificação automática de vitória  
- Identificação de empate ("velha")

**🚢 Batalha Naval**  
Jogo de estratégia utilizando matrizes para:
- Posicionamento de embarcações  
- Realização de ataques  

---

### 🧠 Estruturas de Dados e Conceitos

**📌 Fila (Queue)**  
- Segue o princípio **FIFO (First In, First Out)**  
- Demonstra inserção e remoção de elementos  

**📌 Pilha (Stack)**  
- Baseada no conceito **LIFO (Last In, First Out)**  
- Operações de empilhar e desempilhar  

**📌 Vetor / Matriz**  
- Manipulação de arrays multidimensionais  
- Aplicações práticas com dados  

**📌 Ponteiros**  
- Manipulação de endereços de memória  
- Uso de referências  

**📌 Lista Encadeada**  
- Estrutura dinâmica de armazenamento  
- Inserção e remoção eficiente de elementos  

**📌 Grafos**  
- Representação de vértices e conexões  
- Estrutura base para algoritmos de grafos  

---

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** C++  
- **Bibliotecas Padrão:**  
  - `iostream`  
  - `cstdlib`  
- **Biblioteca Específica:**

## 📋 Como Executar

1. Certifique-se de ter um compilador C++ instalado (ex: GCC / MinGW)

2. Faça o download do arquivo: `Teste.cpp`

3. Compile o código:

`g++ Teste.cpp -o TrabalhoPratico`

4. Execute o programa:
`./TrabalhoPratico`

## 🎮 Como Usar

Ao iniciar o programa, será exibido um menu principal.

Digite o número correspondente à opção desejada:

1 → Fila  
2 → Pilha  
3 → Jogos (Vetor/Matriz)  
4 → Ponteiros  
5 → Lista Encadeada  
6 → Grafos  
0 → Sair
